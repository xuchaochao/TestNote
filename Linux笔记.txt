sudo apt-get update 更新当前系统的软件包。
sudo gedit 某文件名表示用某文本编辑器打开和查看某文件，gedit是GNOME桌面环境下兼容UTF-8的文本编辑器

清屏：clear

dpkg:
dpkg -L package 查看软件安装到什么地方
dpkg -l package 显示包的版本
dpkg -P package 移除软件并且不保留配置

自动补全命令用tab按钮，*表示任意长度的字符串，？表示一个字符。
查看所处目录路径，pwd。
ls命令下，目录为蓝色，普通文件为黑色（或者白色），可执行文件为草绿色，淡蓝色表示这个文件是一个链接文件（相当于快捷方式）。ls -F能显示出目录，ls -a能显示出该目录下所有的隐藏文件,ls -l能查看文件的各种属性。
cat -n 某文件,能以显示行数的方式展示一个文件内容。
more命令能部分展示文件内容，然后按enter键展示更多；当然也可以使用less命令更加的好用。
head，tail -行数 能查看文件的开始或者结束的那几行。
grep 关键词 文件名或者grep ‘关键词（对于有空格的情况用引号）’ 文件名
find 文件名就能找到当前目录下的文件（一般用*？来搭配使用）.
locate *.desktop表示搜索某个文件所在的目录，locate用的是检索文件名数据库来确定文件的位置，如果希望获得该数据库的最新版本那么可以在root权限下用updatedb刷新一下，然后再locate。
